Welcome to Ponder

Ponder is a web application designed to help you generate wordlists.

Usage:

  • Import a wordlist from the server's /import directory or upload a file.
  • Monitor the logs and await the new wordlist.
  • Download a generated wordlist based on the uploaded file.

Latest Log Events

Import Wordlist

Import *.txt files from the /import directory on the server. The file should contain one item per line.

Upload File

Upload a file to generate a wordlist. The file should contain one item per line.

Download Wordlist

Download a generated wordlist based on the uploaded file. The top number of lines will be displayed or downloaded. Include an optional substring to use for filtering.
